# Growth Mechanics Checklist

Analyze retention, acquisition, and engagement mechanisms.

## Core Questions

1. **Activation**
   - What is the aha moment — when does a user first get real value?
   - How quickly is activation reached from sign-up or first use?
   - Is the path to activation clear and frictionless?

2. **Retention**
   - What brings users back after their first session?
   - Are there habit-forming loops or recurring value moments?
   - What would cause a user to stop using the service entirely?

3. **Viral and network effects**
   - Does using the service naturally create shareability?
   - Are there collaboration or social features that expand usage?
   - Does the service get more valuable as more users join?

4. **Engagement loops**
   - Is there a trigger → action → reward → investment cycle?
   - Are re-engagement mechanisms in place (notifications, digests, reminders)?
   - Are there progression mechanics (streaks, milestones, history) that drive return visits?

5. **Referral and organic growth**
   - Is there a referral or invite mechanism?
   - Does the service create outputs or content that drive organic discovery?
   - Are there integration points with platforms users already use daily?

## Priority Signals

Raise to **High** if:
- There is no clear retention mechanism beyond the initial visit
- Activation is delayed, unclear, or blocked by friction
- The service has no natural virality or sharing path

Raise to **Medium** if:
- Retention exists but is not reinforced by habit loops or re-engagement
- Sharing is technically possible but not encouraged or incentivized

Raise to **Low** if:
- Core engagement is solid but growth amplifiers are missing or underdeveloped
