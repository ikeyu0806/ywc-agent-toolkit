# Research Notes

This skill is adapted from the external `ipps-deep-research` skill with Codex-specific simplification.

## Kept

- Primary-source-first hierarchy
- Assumption tracking before research
- Evidence labels for claims
- Structured output with findings, implications, recommendation, and sources

## Removed

- Windsurf-specific credit tracking
- Session orchestration files like STRUT and TASKS as mandatory artifacts
- Tool-specific assumptions that do not map cleanly to this environment

## Codex adaptation

- Keep the workflow lightweight enough to use inside a normal coding task
- Emphasize mapping research back to the local repository
- Prefer concise, reviewable summaries over huge research dossiers unless the user asks for exhaustive output
