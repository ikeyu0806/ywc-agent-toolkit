# Common SaaS Flows

Use this reference when the design is missing important lifecycle paths.

## Account and Membership

- sign up
- invite teammate
- accept invite
- switch tenant
- suspend user
- remove member

## Billing

- start trial
- convert to paid
- upgrade plan
- downgrade plan
- cancel subscription
- recover from failed payment

## Product Operations

- create tenant-scoped resource
- run background processing job
- deliver webhook
- retry failed job
- export customer data
- delete tenant data

## Support and Governance

- impersonation or support access
- audit privileged change
- feature flag rollout by tenant
- rate-limit abusive tenant or user
